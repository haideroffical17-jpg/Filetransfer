How to use QuickShare

  1. Sender: Click "Select File" and choose the file you want to share
  2. Sender: Click "Generate Share Code" to create a 6-digit code
  3. Receiver: Enter the 6-digit code in the receiver section
  4. Receiver: Click "Connect & Receive" to start downloading the file
  5. Works on any device - laptop, mobile, tablet - with any modern browser
